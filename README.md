# LitterBox Frontend

Next.js 14 dApp frontend for the LitterBox Solana program.

## Details

- **Program ID:** `GbDxASiScq4SNjq3Nj5iqYNSkCeyuTLTHSE64pxyAQeD`
- **Network:** Solana Devnet
- **RPC:** Helius Devnet
- **IDL:** `src/lib/litterbox.json` (generated by Anchor 0.32, already included)

## Stack

- Next.js 14 (App Router)
- Tailwind CSS — dark amber/ink design system, Playfair Display + DM Sans
- Framer Motion — page transitions and micro-animations
- @solana/wallet-adapter — Phantom, Solflare, Backpack
- @coral-xyz/anchor 0.32.1 — typed program client from generated IDL
- Jupiter Price API — dust token USD valuation

## Quick Start

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000)

## Views

| View     | Description                                          |
|----------|------------------------------------------------------|
| Overview | Live cycle progress, on-chain config stats           |
| Deposit  | Wallet dust scanner — find and sweep low-value tokens|
| Claim    | Per-cycle $LITTER claim with receipt tracking        |
| Admin    | Buyback trigger + record (authority wallet only)     |

## Testing Deposit

To test the deposit flow you need a devnet SPL token:

```bash
# Create a test mint
spl-token create-token --url devnet

# Create a token account
spl-token create-account <MINT_ADDRESS> --url devnet

# Mint tokens to yourself
spl-token mint <MINT_ADDRESS> 1000 --url devnet
```

Connect your wallet, go to Deposit, and your test token should appear in the dust scanner
(price will be $0.00 since it has no Jupiter listing, which puts it below the $1 threshold).

## Deploying to Vercel

```bash
yarn build
vercel --prod
```

Set these environment variables in Vercel:
- `NEXT_PUBLIC_RPC_URL` — your Helius or QuickNode RPC endpoint
- `NEXT_PUBLIC_PROGRAM_ID` — `GbDxASiScq4SNjq3Nj5iqYNSkCeyuTLTHSE64pxyAQeD`
